## 📌 Overview

An in-depth data analysis project uncovering the key drivers of delivery delays in Delhivery's vast logistics network. Combines EDA, statistical testing, and visualisation to deliver actionable operational insights from real-world logistics data.

**Tech Stack:** `Python` `Pandas` `Seaborn` `EDA` `Logistics` `Supply Chain`

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install -r requirements.txt
```

> Requires **Python 3.8+**

### Installation

```bash
git clone https://github.com/Onabanjomicheal/Delivery-Logistics-Data-Analysis.git
cd Delivery-Logistics-Data-Analysis
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
